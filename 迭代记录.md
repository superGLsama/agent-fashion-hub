## 2026-04-20 - 首次优化迭代

### 优化内容

**目标**：基于网站搭建学习知识，增强搜索/筛选体验和交互动效

### 具体改动

1. **搜索功能**
   - 添加搜索栏（支持名称/描述/标签搜索）
   - 250ms 防抖输入，避免频繁渲染
   - 搜索词在名称中青绿色高亮（正则转义防 XSS）
   - 右上角清除按钮，Esc 键清空
   - 搜索状态持久化到 localStorage

2. **稀有度筛选**
   - 新增稀有度筛选栏（全部/传奇/史诗/稀有/普通）
   - 每个等级有专属颜色高亮
   - 与联盟筛选、搜索词三者叠加生效

3. **卡片入场动画**
   - 每张卡片 `cardIn` 动画入场
   - 交错延迟（index * 60ms），依次展开

4. **骨架屏加载态**
   - 初始化时显示 6 个骨架卡片
   - Shimmer 闪烁动画，用户感知加载进度

5. **积分消耗动画**
   - 领取装备时积分数字平滑递减
   - `requestAnimationFrame` + easeOut 缓动

6. **空状态友好提示**
   - 无结果时显示当前搜索词
   - 给出针对性建议和清除筛选按钮

## 2026-04-27 - Day 13 迭代

**目标**：参考 Agent Street 迭代方向，为潮流平台添加搜索历史、热门推荐和收藏夹系统

### 具体改动

- **导航栏收藏按钮**：右上角新增❤️收藏按钮，显示收藏数量角标

- **快速操作**：收藏面板支持从收藏快速购买装备

- **智能提取**：从所有装备/Agent/装扮名称中随机抽取

- **粉红主题**：与搜索历史区分，使用火焰emoji和粉色主题

#### 4. 交互优化
- **聚焦自动展开**：搜索框聚焦时自动显示建议面板
- **输入自动隐藏**：输入内容时自动隐藏，避免干扰
- **Esc键关闭**：关闭面板并取消聚焦
- **装扮页面收藏**：装扮系统的服装卡片也支持收藏

- 面板遮罩层防止背景点击穿透

## 2026-05-XX - Day XX 迭代 - 商品卡片3.0升级

### 迭代内容

**目标**：借鉴游戏UI设计，让商品卡片更具吸引力和互动性

### 具体改动

#### 1. 稀有度光效系统
- **传奇装备**：金色脉冲光效，3秒周期，带有内发光效果
- **史诗装备**：紫色渐变光效，2.5秒周期
- **稀有装备**：蓝色微光，2秒周期
- **普通装备**：基础边框，无特效
- **渐变边框**：使用CSS mask实现渐变边框效果

#### 2. 3D悬浮倾斜效果
- **鼠标跟踪**：根据鼠标在卡片上的位置动态计算倾斜角度
- **透视变换**：使用 perspective(1000px) 创建3D空间感
- **角度限制**：rotateX/Y 限制在 ±15 度，防止过度倾斜
- **平滑过渡**：150ms ease-out 响应速度
- **悬浮提升**：hover时自动上浮8px并放大1.02倍

#### 3. 卡片入场交错缩放淡入
- **弹性缓动**：cubic-bezier(0.34, 1.56, 0.64, 1) 弹性曲线
- **交错延迟**：每张卡片间隔80ms入场
- **缩放效果**：从scale(0.9)弹到scale(1.02)再回归1
- **淡入效果**：opacity 0→1 + translateY(30px)→0

#### 4. 点击粒子反馈
- **稀有度配色**：粒子颜色与装备稀有度对应
- **粒子数量**：每次点击生成12-20个粒子
- **散射动画**：粒子沿圆形轨迹向外散射，带有向上偏移
- **尺寸随机**：粒子大小4-12px随机
- **发光效果**：粒子带有box-shadow发光

#### 5. 收藏按钮动效升级
- **心跳动画**：已收藏状态持续心跳效果（1.5秒周期）
- **悬停旋转**：hover时旋转15度并放大
- **快速心跳**：hover已收藏按钮时心跳加速

### 技术实现

```javascript
// 3D倾斜核心逻辑
function handleCardTilt(e, card) {
    const rect = card.getBoundingClientRect();
    const rotateX = ((e.clientY - rect.top) / rect.height - 0.5) * -30;
    const rotateY = ((e.clientX - rect.left) / rect.width - 0.5) * 30;
    card.style.transform = `perspective(1000px) rotateX(${rotateX}deg) rotateY(${rotateY}deg)`;
}

// 粒子效果核心逻辑
function createParticles(e, rarity) {
    const particleCount = 12 + Math.floor(Math.random() * 8);
    for (let i = 0; i < particleCount; i++) {
        const angle = (Math.PI * 2 * i) / particleCount;
        const distance = 50 + Math.random() * 80;
        particle.style.setProperty('--tx', `${Math.cos(angle) * distance}px`);
        particle.style.setProperty('--ty', `${Math.sin(angle) * distance}px`);
    }
}
```
